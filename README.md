# ADB-Tool For Windows!
_____________________________________________________
---How to install---

1. Download zip file "ADB-Tool-master.zip"
2. Unzip "ADB-Tool-master.zip" to Desktop
3. Open ADB-Tool
4. Unzip "adb.zip" in ADB-Tool
5. Start "ADB-Tool.bat"
